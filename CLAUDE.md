## Architecture

React 19 + TypeScript + Vite + Tailwind CSS 4. Deployed to GitHub Pages (main branch → `/PokepelagoClient/`, beta branch → `/PokepelagoClient/beta/`).

This is a companion web client for the Archipelago Pokepelago APWorld randomizer. Players connect to an Archipelago multiworld server and guess Pokemon names to "catch" them, with lock gates controlling which Pokemon are guessable.

### State Management

Two React contexts manage all state — no external state library:

- **GameContext** (`src/context/GameContext.tsx`) — Central game state: caught/unlocked Pokemon, lock gates (gym badges, daycare, fossils, trade evos, etc.), items, connection state, UI settings, game mode (Archipelago vs Standalone). Exposes `checkPokemon()`, `unlockPokemon()`, `isPokemonGuessable()`, `connect()`, `disconnect()`, `say()`.
- **TwitchContext** (`src/context/TwitchContext.tsx`) — Twitch chat integration: leaderboard, credits, guess feed. Uses BroadcastChannel for multi-tab sync and IndexedDB for persistence.

### Key Hooks

- **useAPConnection** (`src/hooks/useAPConnection.ts`) — WebSocket connection to Archipelago server via `@airbreather/archipelago.js`. Handles auth, item/location packets, hints, connection quality monitoring, ping latency.
- **useOffsets** (`src/hooks/useOffsets.ts`) — Manages AP item/location ID offset tables. Detects legacy vs new AP world versions.
- **useGuessEngine** (`src/hooks/useGuessEngine.ts`) — Multi-language Pokemon name matching (11 languages). Handles special cases (Nidoran-F/M, Mr. Mime, Ho-Oh, etc.) with accent/diacritic stripping.
- **useTrapHandler** (`src/hooks/useTrapHandler.ts`) — Processes trap items (derp sprites, releases, shuffles).
- **useSpriteManager** (`src/hooks/useSpriteManager.ts`) — Resolves sprite URLs from multiple sources (PokéAPI, custom imports, derpemon, PMD).
- **useTwitchChat** (`src/hooks/useTwitchChat.ts`) — Connects to Twitch TMI chat, monitors for Pokemon guesses with rate limiting.

### Data Layer

- `src/data/pokemon_metadata.json` — Static metadata for all 1025 Pokemon (name, types, legendary status).
- `src/data/pokemon_names.json` — Multi-language name lookup (11 languages). Generated by `scripts/generate_pokemon_names.py`.
- `src/data/pokemon_gates.ts` — Lock gate Pokemon ID lists (legendaries, babies, fossils, trade evos, ultra beasts, paradox, stone evos). Must stay in sync with the Archipelago Pokepelago APWorld's `data.py`.

### Guess Flow

1. Input comes from `GlobalGuessInput` (keyboard) or `useTwitchChat` (Twitch chat)
2. `useGuessEngine` matches input against `pokemon_names.json` across all enabled languages
3. `GameContext.isPokemonGuessable()` evaluates all active lock conditions (type keys, gym badges, region locks, etc.)
4. Valid guess → `checkPokemon()` updates state, sends AP location check, logs event

### Build Configuration

`vite.config.ts` uses environment variables for deployment targets:
- `DEPLOY_TARGET=beta` → base path `/PokepelagoClient/beta/`
- `GITHUB_ACTIONS=true` → base path `/PokepelagoClient/`
- Local dev → base path `/`
- `__IS_BETA__` and `__TWITCH_ENABLED__` are compile-time defines

### Storage

- **localStorage** — Profiles, UI settings, game state persistence
- **IndexedDB** — Custom sprites (via `idb` library), Twitch leaderboard/credits
